## Description
Nexus Interactive 3D Portfolio is a modern web application designed to present digital portfolios in an immersive and interactive 3D environment. Built with React and Vite, it offers a dynamic platform for showcasing projects, skills, and professional roles through engaging visual experiences. This project serves as a central hub (NexusHub) for all your digital creations, allowing visitors to explore your work with a unique interactive perspective.

## Features
*   Interactive 3D scenes for dynamic project and role showcasing.
*   Modular component structure for easy scalability and maintenance.
*   Dedicated sections for project details and role descriptions (e.g., `RoleDetail`).

## Installation
Follow these steps to set up the project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/[your-username]/nexus-interactive-3d-portfolio.git
    cd nexus-interactive-3d-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or yarn install
    ```

3.  **[Optional] Configure environment variables:**
    If your project requires any API keys or specific configurations, create a `.env` file in the root directory and add them as `VITE_YOUR_VARIABLE_NAME=your_value`.

## Usage
To run the development server:

```bash
npm run dev
# or yarn dev
```

This will start the application, typically accessible at `http://localhost:5173`.

To build the project for production:

```bash
npm run build
# or yarn build
```

This will generate a `dist` directory with the optimized production build.
